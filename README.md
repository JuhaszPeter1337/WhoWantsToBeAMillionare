# Who Wants To Be A Millionare game with Pygame

## Description

Who Wants To Be A Millionare game in Python

## Prerequisites

### python3

```
sudo apt update
sudo apt install python3 -y
```

### pip3

```
sudo apt install python3-pip -y
```

### pygame

```
pip3 install pygame
```

## Start app

```
python3 main.py
```
